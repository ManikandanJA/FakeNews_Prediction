About This Project

A Machine Learning powered Fake News Detection System

📌 Project Overview

The Fake News Prediction System is a web application built using Python (Flask) and Machine Learning techniques to identify whether a given news article or headline is Real or Fake.

The system uses two approaches: a Trusted Facts Database for verified facts, and a TF-IDF + Passive Aggressive Classifier ML model for general predictions.

🧠 How the ML Model Works

1
Data Preprocessing

News text is cleaned and prepared for analysis.

2
TF-IDF Vectorization

Text is converted into numerical feature vectors using Term Frequency-Inverse Document Frequency.

3
Passive Aggressive Classifier

An online learning algorithm that is well-suited for large-scale text classification.

4
Confidence Score

Decision function output is converted to a probability score using sigmoid function.
